# BrainC
### Brainfuck language interpreter written in C for learning purposes
---
## Example of usage
- Open terminal and go to *brain-fuck-interpreter* directory.
- Use **./bin/linux/brainc ./brainfuck-examples/hello-world.bf** command. </br>
If path argument is not specified, interpreter assumes *./main.bs* 
as its path.
